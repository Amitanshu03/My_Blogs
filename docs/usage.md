

# 🧑‍💻 Usage Instructions

This blog project is purely static.

## 🌐 Live Hosting

You can host it using:
- GitHub Pages
- Netlify
- Vercel

### GitHub Pages Steps:
1. Go to the repo → Settings
2. Scroll down to **Pages**
3. Choose branch: `main` and folder: `/ (root)`
4. Hit Save
